**[rag_cpu](https://github.com/AdityaSharma2457/rag_cpu)** 🎧 *(Group Project)*
An AI-powered counselling call intelligence engine — transcribes, diarizes, fact-checks, and scores university admission counselling calls, running entirely on CPU.
- **Pipeline:** WhisperX (CPU, int8) transcribes & diarizes → LLM extracts factual claims → Chroma vector store fact-checks them against real brochures → RAG engine scores the counsellor's performance and generates a markdown report
- Scores counsellors on professionalism, empathy, objection handling, and more — plus a follow-up chat feature to ask questions about a completed report
- **Stack:** Flask, WhisperX, LangChain, ChromaDB (MiniLM embeddings), NVIDIA Nemotron & Gemini, SQLAlchemy, Docker

**[VendHub-Ecommerce-App](https://github.com/ankitpyj/VendHub-Ecommerce-App)** 🛒
A full-stack platform connecting farmers directly to vendors, cutting out middlemen for fairer trade.
- Role-based access: **Farmer** (list/manage products, view orders), **Vendor** (browse, cart, order), **Admin** (user & approval management)
- **Frontend:** HTML, CSS, JavaScript · **Backend:** Node.js, Express.js
- Planned next: MongoDB integration, authentication, payment gateway, real-time order tracking, mobile-friendly UI
